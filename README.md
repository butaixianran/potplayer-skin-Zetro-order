# potlayer皮肤:Zetro order
一个老旧但好用的potplayer皮肤。

鼠标放到各个按钮图标上面，会显示这个按钮的 左键点击功能|右键点击功能。

因为每个按钮有2个功能，常用功能齐全，所以特别方便。

皮肤太旧，不再更新。

## 使用方法
在release中，下载Zetro_order.dsf 文件，放到你的Potplayer安装目录的skins目录下即可在potplayer->菜单->皮肤 中切换此皮肤。
